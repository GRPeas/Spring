jekyll 
----
https://iingang.github.io/posts/windows-github-set/


1. 루비 설치<br>
https://rubyinstaller.org/downloads/

2. 명령어 입력<br>
   루비 명령어 프롬프트에서 
   
```
gem install jekyll
gem install minima
gem install bundler
gem install jekyll-feed
gem install tzinfo-data
```
1. 설치 확인
```
jekyll -v
```

4. Repository 클론 및 url 복사
   
5. 연결
```
 [사용자명].github.io
 ```

6. 클론한 폴더에서 명령어 입력
```
gem install tzinfo
gem install tzinfo-data
bundle install
bundle update
bundle install
```

7. 실행
```
bundle exec jekyll s
```

8. 에러 발생시
- 
```
vi Gemfile
```
- 내용 추가
```
gem "tzinfo"
gem "tzinfo-data"
```
- 재실행
```
bundle exec jekyll s
```
